# markdown について

- markdown記法は、シンプルな記述方法です。
- html にも変換できます。


## 図の挿入

- 以下のように記載すると、図やリンクを挿入できます。

```
![](図のパス)

例1）
![](./人類が勝利しました.jpg)

例2）
![人類が勝利しました](./人類が勝利しました.jpg)
```

![](./人類が勝利しました.jpg)


## 表

- 以下のように記載すると、表を書くことができます。

```
| # |lnk|
|---|---|
| 1 |http://www.fmu.co.jp/redpillradio.html|
| 2 |https://qajf.github.io|
```
